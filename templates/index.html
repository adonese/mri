PDS is a service to detect pneumonia depending on a deep learning classification model which is built based on more than 5000 X-ray images from Large Dataset of Labeled Optical Coherence Tomography (OCT) and Chest X-Ray Images by Daniel Kermany et.al. However pneumonia is a good indicator in the identification of novel coronavirus affected cases.

This work is built on this paper view the paper and the dataset. You can also download the model and view the source code at the project github repo.

# Why ?

This service can be used to help doctors in diagnosis what does mean saving time and better performance for health system however this service is tested on more than 300 X-ray image to get an accuracy of 98% what means these service can be trusted to help doctors in diagnosis as well as reducing the stress upon countries health system during coronavirus pandemic.

# Background

Multiple researchers used deep learning convolutional neural network for the purpose of classification for Chest X-ray to detect pneumonia and get good test accuracy( Saraiva et al (95.30 %),G. Liang and L. Zheng(96.70%), H. Wu et al (97%) and Nour EldeenM. Khalifa et al who used Generative Adversarial Neural Network to Generate dataset from previously mentioned dataset and used these generated data to reach an accuracy of (99.0%) while using Resnet18 model. In our service we bring the results of these researches to the light by offering this solution as an easy to use service to satisfy doctors needs while standing on solid scientific knowledge.

# About model

VGG16 model is used with transfer learning by keeping feature extraction wights while adding 3 fully connected layers which trained on previously mentioned dataset and found that test accuracy is 98% .
