PDS is a service to detect pneumonia depending on a deep learning classification model which is built based on more than 5000 X-ray images from Large Dataset of Labeled Optical Coherence Tomography (OCT) and Chest X-Ray Images by Daniel Kermany et.al. However pneumonia is a good indicator in the identification of novel coronavirus affected cases.

This work is built on this paper view the paper and the dataset. You can also download the model and view the source code at the project github repo.
